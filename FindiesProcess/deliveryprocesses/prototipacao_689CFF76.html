Task: Prototipacao
Essa atividade irá trazer uma melhor forma de interação dos usuários com o projeto, prevenindo erros e trazendo feedbacks constantes, uma vez que a atividade deverá ser executada diversas vezes, para o refinamento do protótipo.
Purpose
A prototipação precisa ser realizada de forma simples.
Relationships
RolesPrimary:
  • front-end
Additional: Assisting:
Main Description

A prototipação de uma interface em papel é mais rápida e, além de economizar tempo,
é mais fácil e rápido para ser ajustado. O objetivo final da prototipação é obter um
protótipo de baixo nível validado e pronto para o desenvolvimento.

Os usuários irão dar feedbacks constantes nos protótipos mostrados. O
número de vezes e o tempo que se passará nessa atividade irá depender de cada
projeto e, principalmente, do desejo do usuário.
